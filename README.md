# Flutter Shop Application 🛒

A mobile shopping application built with Flutter that displays products dynamically using REST API integration.

## Features
- User Registration
- User Login Authentication
- Products Grid View
- Product Details Page
- API Integration
- Provider State Management
- Modular Code Structure
- Reusable Custom Widgets

## Technologies Used
- Flutter
- Dart
- REST API
- Provider
- OOP Concepts

## Screens
- Sign Up Page
- Login Page
- Products Page
- Product Details Page
